## Usage

```python
#install Composer
composer install

#copy env
cp .env.example .env

#generate key
php artisan key:generate

#setup your db in .env
DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=your_db_name
DB_USERNAME=username if you set
DB_PASSWORD=password if you set

#migration table
php artisan migrate or php artisan migrate:fresh
```
